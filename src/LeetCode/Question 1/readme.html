In this task, a prototype for a text editor application is to be implemented consisting of the following functionalities:

Command Action
["Insert", s]

Insert the string at the current cursor position,

the cursor will also move to the right by the length of the inserted string
["Print", x] If the cursor is at position m, print all the characters from m - x to m + x
["Left", x] Move the cursor x positions to the left
["Right", x] Move the cursor x positions to the right
["Backspace", x] Remove x characters to the left of the cursor
["Delete", x] Remove x characters to the right of the cursor

 

Given a 2-d array of strings size n x 2, commands, find the string obtained on each of the print operations by performing the above operations starting from an empty string and cursor at the starting position and report them as an array of printed strings in order. If for any operation the cursor reaches the beginning or the end of the current string, it does not go any further on the either side.

 

Note: For consistency, the second argument is always provided as a string and hence might be required to be converted to an integer for the relevant commands.

 

Example

Suppose n = 6 and commands = [["Insert", "hello"], ["Left", "3"], ["Delete", "1"], ["Right", "2"], ["Backspace", "1"], ["Print", "4"]].

 

Command Current String Current Cursor Final String Final Cursor Print
Insert "hello" "" 0 "hello" 5 (after o)  
Left 3 "hello" 5 "hello" 2 (after e)  
Delete 1 "hello" 2 "helo" 2 (after e)  
Right 2 "helo" 2 "helo" 4 (after o)  
Backspace 1 "helo" 4 "hel" 3 (after l)  
Print 4         Print characters from 3 - 4 to 3 + 4. Hence print "hel"

 

Hence the answer is ["hel"]

 

Function Description

Complete the function getPrintedStrings in the editor below.

getPrintedStrings has the following parameter(s):

    commands[n][2]:  the commands to be performed

 

Returns

    string[]: the printed strings

 

Constraints

  • 1 ≤ n ≤ 5000
  • The sum of lengths of all strings inserted does not exceed 5 * 105
  • The size of the first string inserted does not exceed 100 and the length of all other strings inserted does not exceed 25.
  • All the strings consist of lowercase English letters.

 

Input Format For Custom Testing

The first line contains an integer, n, the number of rows in commands.

The following line contains a constant integer, 2, the number of columns in commands.

Each of the next n lines contains two space-separated strings, commands[n][0] and commands[n][1].

Sample Case 0

Sample Input For Custom Testing

STDIN                 FUNCTION
-----                 --------
7                →    commands[] size n = 7
2                →    commands[] size const = 2
Insert addthis   →    commands = [["Insert", "addthis"], ["Print", "5"], ["Left", "4"], ["Right", "2"], ["Backspace", "1"], ["Delete", "1"], ["Print", "10"]]
Print 5
Left 4
Right 2
Backspace 1
Delete 1
Print 10

Sample Output

dthis
addts

Explanation

After each operation, the string and the position of the cursor are as follows:

  1. "addthis", cursor after 7
  2. Prints characters from 7 - 5 = 2 to 7 + 5 = 12 i.e. "dthis"
  3. Cursor moves to 3
  4. Cursor moves to 5
  5. The string becomes "addtis", and the cursor moves to 4
  6. The string becomes "addts", and the cursor remains at 4
  7. Prints string from -6 to 16 i.e. "addts"
